Chapter 2-Installation

Installing VS Code

For Windows

  1. Download the Visual Studio Code installer from the official website
  2. Run the installer and follow the on-screen instructions.
  3. Once installed, open VS Code.

For Mac:

  1. Download the Visual Studio Code installer from the official website
  2. Open the downloaded .dmg file
  3. Drag the VSCode icon to the applications folder
  4. open VSCode from the applications folder

Setting up MinGW Compiler:

For Windows:

MinGW (Minimalist GNU for Windows) provides a set of freely available and freely distributable compilers.

  1. Download the MinGW installer from the official website
  2. Run the installer and select the components you want to install. Make sure to include the C and C++ compilers.
  3. During installation, choose the installation directory. Note this directory for later use.
  4. Add the MinGW bin directory to your system's PATH environment variable. This allows VS Code to find the compiler. To do this:
    • Right-click on "This PC" or "Computer" on your desktop or in File Explorer.
    • Click on "Properties."
    • Click on "Advanced system settings."
    • Click on "Environment Variables..."
    • Under "System variables," find the "Path" variable and click "Edit..."
    • Add the path to the MinGW bin directory (e.g., C:\MinGW\bin).
  5. Open a new command prompt and test the installation by typing gcc --version and g++ --version. You should see version information.

For Mac:

On Mac, you can use the Xcode Command Line Tools, which include the GCC compiler.

  1. Open Terminal
  2. Run the following command to install the Xcode Command Line Tools: xcode-select --install
    Follow the on-screen instructions to complete the installation
  3. Test the installation by typing gcc --version in the Terminal. You should see version information.
